Haskell solutions to Project Euler problems.

[![Build Status](https://travis-ci.org/whittle/euler.svg?branch=master)](https://travis-ci.org/whittle/euler)

Tested with the following versions:
  * GHC 7.10.1
  * cabal-install 1.22.0.1
  * LAPACK 3.5
  * BLAS 1.2
